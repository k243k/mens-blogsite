# PROJECT BRIEF

- **事業:** メンズエステ体験レビューサイト（レビューDB型・有料本音レビューあり）
- **種別:** 既存サイトのUI全面リニューアル（世界観ごと変更OK・1からやり直し許可済み）
- **目的:** 読者が「この店に行くべきか」を判断できる。無料で信頼を作り、有料本音レビューへ誘導
- **最重要CTA:** 記事を読む → 有料ロック解除（1記事300〜500円想定）
- **ターゲット:** 店選びで失敗したくない男性・メンエス初心者・写真と実物のギャップを知りたい人

## サイトの魂（2026-07-03 ヒアリングで確定）

> 「最後は経験を売ります。私が今までやってきた経験を元に、あなたを失敗させない」

- 売り物は情報の網羅性ではなく **筆者の経験値**。
- ポジション＝「先に失敗してきた案内人」。読者を失敗させないための判断材料を出す。
- デザインは案内人の人格・経験の蓄積が見える形にする（匿名の情報サイトにしない）。

## 現UIの不満（全項目該当）

1. 見にくい・読みにくい（暗くて文字が読みづらい）
2. 興味がそそられない（クリックしたくならない）
3. AIっぽい・既視感（テンプレ感、固有の顔がない）
4. 情報が探しにくい（エリア・スコア・料金で辿り着けない）

## 技術制約（変更しない）

- Next.js 15 + TypeScript + Tailwind + Supabase / 静的export / 固定費0円方針
- ルーティング・データ層（repository）は既存を流用。刷新はUIレイヤー
- 実画像なし。プレースホルダー＋後日AI生成（ChatGPT/Playwright経由）差し替え
